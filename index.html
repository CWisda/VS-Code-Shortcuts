Useful Keyboard Shortcuts

  1. Ctrl + S - Save
  2. Ctrl + C - Copy
  3. Ctrl + V - Paste
  4. Arrow Keys for cursor movement
  5. Shirt + Arrow Direction to select
  6. Alt + Arrow Up/Down - move entire line up/down

Multi-cursor Operations

  • Ctrl + D - select multiple of the same thing
  • Alt + left click to generate mulitple cursors
  • Find + Replace
